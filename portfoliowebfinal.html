CompSci Final

For my CompSci Final I worked with a partner and we created a trivia game/race. Players must answer questions racing against another person. The first person to finish wins. Here are some screen shots of the game and the code for it below!




Here's the code for the game. We had to make two other files besides the main one to make it easier on ourselves. We created a text file which another file reads from. That way it is easier to edit our questions and answers as well as add information. The second file we created was a reading file which interpreted the text file. This deciphers which line is an answer or a question and puts them into categorized lists. The main file takes the lists and chooses random items from the list(the questions) by using the random.randint function. We programed the game to loop on the same question until the user answers correctly.

    questions = []
    answers = []
    hquestions = []
    hanswers = []
    counter = 1
    easy = True
    with open("file.txt", "r") as ins:
        for line in ins:
            if line.rstrip('\n') == "HARD QUESTIONS":
                easy = False
                #print "HARD QUESTIONS START HERE: " + str(counter)
                counter+=1 #to adjust it? maybe?
                continue
            if counter%2 == 1:
                if easy:
                    questions.append(line.rstrip('\n'))
                else:
                    hanswers.append(line.rstrip('\n'))
            else: 
                if easy:
                    answers.append(line.rstrip('\n'))
                else:
                    hquestions.append(line.rstrip('\n'))
            counter += 1
        return questions, answers, hquestions, hanswers
